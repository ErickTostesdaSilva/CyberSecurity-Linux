# CyberSecurity-Linux 

Este repositório é focado em projetos práticos de segurança cibernética e administração de sistemas Linux.

## Projeto: Gestão de Permissões de Arquivos
Neste laboratório, realizei a auditoria e modificação de permissões em um ambiente Linux para garantir a segurança de dados organizacionais.

### O que foi praticado:
* **Auditoria:** Uso do comando `ls -la` para identificar vulnerabilidades de acesso.
* **Controle de Acesso:** Manipulação de permissões para Utilizadores, Grupos e Outros com `chmod`.
* **Organização:** Gestão de diretórios sensíveis e arquivos ocultos.
